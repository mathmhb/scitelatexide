The perl scripts in this folder were written for use with MiCTeX bundle. They are not essential part of the SciTE LaTeX IDE source tree, but can be used in juction with SciTE LaTeX IDE to make it easy in doing latexing.

FontsGen.pl is a script for generating necessary Chinese font configurations.

SmartTeXer.pl is a script which makes typesetting in latex (and related formats) automatic, i.e. users no longer need to take care about which engine to use and which route to go through in order to get a final output.
